# Sql-Data-Analysis

A collection of SQL-based data analysis projects focused on extracting insights from structured datasets using PostgreSQL.

---

## Project Overview
This repository contains multiple data analysis projects performed using SQL in PostgreSQL, with Valentina Studio as the primary environment. Each project follows a structured workflow from raw data preparation to advanced querying and insight generation.

The main objective is to transform raw CSV datasets into relational databases and analyze them using efficient SQL techniques.

---

## Workflow

Each project follows a consistent step-by-step process:

1. **Data Collection**  
   - Raw data is obtained in CSV format  

2. **Database Creation**  
   - A PostgreSQL database is created  
   - Tables are designed based on dataset structure  

3. **Table Design & Constraints**  
   - Columns are defined with appropriate data types  
   - Constraints such as PRIMARY KEY, FOREIGN KEY, and NOT NULL are applied  

4. **Data Import**  
   - CSV data is loaded into the created tables  

5. **Data Analysis**  
   - SQL queries are written to analyze the data  
   - Techniques include:
     - JOIN operations  
     - Aggregate functions  
     - Subqueries  

6. **Insight Generation**  
   - Analytical thinking is applied to interpret results  
   - Key findings and data-driven decisions are derived

## Featured Project

### Cybersecurity Risk Analysis
- Designed a relational database including users, systems, organizations, login logs, and security events  
- Analyzed authentication patterns and system activity to detect potential risks  
- Identified unusually high login failure rates (~50%), indicating possible systemic authentication issues  
- Discovered that failed login attempts are widely distributed, suggesting non-targeted risk patterns  
- Highlighted elevated risk in Admin accounts and sensitive industries such as Finance and Healthcare  
- Analyzed incident distribution across operating systems and organizations  
- Demonstrated that security risks are multi-layered and not limited to a single factor
